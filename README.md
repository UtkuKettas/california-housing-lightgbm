# california-housing-lightgbm

## Project Overview
This project predicts California housing prices using LightGBM regression.

## Dataset
California housing dataset with demographic and geographic features.

## Data Preprocessing
- Missing values filled using median
- Outliers handled using transformation
- Categorical variables encoded
- Feature correlations analyzed

## Model
LightGBM Regressor

## Results
R2 Score: 0.83
